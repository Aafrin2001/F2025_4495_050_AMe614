Project Structure

The EAi mobile app is built using React Native with Expo, providing a cross-platform solution for iOS and Android. The architecture follows a single-page application (SPA) pattern with centralized navigation control.

Key Technical Decisions:

  • Expo Framework: Chosen for rapid development and easy deployment
  • TypeScript: Full type safety throughout the application
  • Centralized Navigation: Single App.tsx controls all screen transitions
  • Component-Based Architecture: Each screen is a self-contained React component
  • State Management: React hooks (useState) for local state management
