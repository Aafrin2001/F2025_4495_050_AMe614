🔧 Figma Implementation Tips

Setting up your Figma file:

  1. Create a new Figma file with mobile frame (375x812px)
  2. Set up your color palette as Figma styles
  3. Create text styles for each typography scale
  4. Build components for buttons, cards, and icons
  5. Use auto-layout for responsive components
  6. Create variants for different states (hover, pressed, disabled)
// Figma Auto-Layout Settings // Buttons: Horizontal, 12px padding, 8px gap // Cards: Vertical, 20px padding, 8px gap // Grid: 2 columns, 15px gap, fill container // Text: Auto-width, 1.6 line height
